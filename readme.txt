Ce projet est une analyse exploratoire qui vise à appliquer les compétences acquises aussi dans ma formation que sur mon temps personnel.
Le data représente l'ensemble des décès humains cans le monde par pays et par cause de l'année 1990 à l'année 2019.

Lien du dataset : https://www.kaggle.com/datasets/madhurpant/world-deaths-and-causes-1990-2019
