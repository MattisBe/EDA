Ce projet est une analyse exploratoire qui vise à appliquer les compétences acquises aussi bien dans ma formation que sur mon temps personnel.
Le dataset représente l'ensemble des décès humains dans le monde par pays et par cause, de l'année 1990 à l'année 2019.

Lien du dataset : https://www.kaggle.com/datasets/madhurpant/world-deaths-and-causes-1990-2019
